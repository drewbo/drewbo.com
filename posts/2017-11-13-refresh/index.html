Blog Refresh

2017.11.13

I've done a pretty bad job of updating my blog for the past three years: it's pretty hard to run from a technical perspective and my "technical side projects" largely became my "job". But I'm starting it back up again.

The site is now written with Markdown instead of reStructuredText. The CSS is much smaller and all my own code as opposed to a theme. It's built with Metalsmith instead of Pelican. The build is done automatically via CircelCI, replacing my previous manual (error-prone) process.
